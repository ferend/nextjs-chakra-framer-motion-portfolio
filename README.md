# Next.js, Chakra UI, Framer Motion, Three.js Portfolio

My personal portfolio website built with Next.js, Chakra UI, Framer Motion, and Three.js. This project showcases my skills, projects, and provides an interactive experience for visitors.

## Technologies Used

- **Next.js:** A React framework for building efficient and scalable web applications.
- **Chakra UI:** A simple, modular, and accessible component library for React.
- **Framer Motion:** A motion library for React that makes it easy to create stunning animations.
- **Three.js:** A lightweight and cross-browser JavaScript library/API for creating 3D graphics.

## Features

- **Responsive Design:** The portfolio is designed to be accessible and look great on various devices.
- **Interactive Animations:** Framer Motion is utilized to create smooth and engaging animations.
- **3D Visualizations:** Three.js is employed to add dynamic and interactive 3D elements.

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.
